# ViterbiSystemC
**Dev Env: Ubuntu repo 14.04LTS**

This project implements the Convolutional Encoder and Viterbi Decoder using SystemC

- Install cmake.

- cd build

- cmake .

- make

The built program name viterbi, run the file to generate the .vcd file for tracing purpose.
